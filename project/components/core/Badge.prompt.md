**Badge** — semantic availability/status pill with a leading dot. Use on cat cards and profiles to show adoption state.

```jsx
<Badge status="available">Örökbe fogadható</Badge>
<Badge status="reserved">Foglalt</Badge>
<Badge status="urgent">Sürgős</Badge>
```

Statuses: `available` (sage green), `reserved` (amber), `urgent` (terracotta), `info` (dusty blue), `neutral`. Sizes `sm`/`md`.
